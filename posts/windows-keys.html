windowskey激活

这个礼拜把工作站正式从 windows10 迁移到了 Fedora24，感觉还不错，就是工作需要必须运行 QQ，虚拟机折腾了不少时间，还好 vbox 支持物理驱动器虚拟化，省了我不少硬盘空间。

下面是激活步骤：

; windows 10 激活
;
; 更新 key
slmgr.vbs /ipk {keys}
; 设置激活服务器
slmgr.vbs /skms {server}
; 激活
slmgr.vbs /ato
; 查看超期时间
slmgr.vbs /xpr

; office16 激活
cd C:\Program Files\Microsoft Office\Office16
cscript OSPP.VBS /sethst:{server}
