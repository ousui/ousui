vmware fusion 网络配置

2017 年 03 月 30 日
虚拟机 macos
1 min read

之前一直是用开源的 virtualbox，后来发现某些 usb 设备例如 u 盾无法挂载，资源无法分配，然后试了 pd 和 vmware, 都可以正常使用。
无语，故换之。 经过对比后，选择了 vmware，还好虚拟机互转还方便。 安装后发现了 vmware 的网络配置这里的一些问题，特地记录一下。

  • macos 下，ui 上改不了网络配置，需要在 “/Library/Preferences/VMware Fusion” 目录下更改。
  • 默认网卡配置 vmnet1/only host，vmnet8/nat 都可以修改，修改对应 ip 就可以。

  • 有时候要用到 ssh 代理跳转，例如某些特定的 vpn 只有 windows 版本，那么在 dhcpd.conf 文件下新增

    host windows-xp {
  hardware ethernet 00:**:**:**:**;
  fixed-address 172.20.33.*;
}
